# Cloud Automation Express App

## Setup
1. Install Node.js+npm from https://nodejs.org
2. In this folder, run:
   ```bash
   npm install
   npm start
   ```
3. Open http://localhost:3000

## Files
- `app.js` - Express server with Bootstrap 5 homepage
- `package.json` - Project metadata and start script
- `index.html` - Existing static page (unused by Express)
